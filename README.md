# Stock Tracker Web Terminal

## 快速启动

```bash
cd /root/stock_tracker && docker-compose up -d
```

浏览器访问：http://<你的NAS IP>:20001

## 功能列表

| # | 程序 | 说明 |
|---|------|------|
| 1 | 主力资金流向博弈图 | 散户/机构/北向三方对战实时模拟 |
| 2 | 涨停敢死队模拟器 | 体验打板快感，35%概率当日亏20% |
| 3 | K线跑酷游戏 | 你是蜡烛图，在K线丛林里跑酷 |
| 4 | 财报排雷大师 | 扫雷游戏，但扫的是财报地雷 |
| 5 | 板块轮动窥探器 | 雷达图展示资金轮动节奏 |
| 6 | 解套倒计时 | 输入成本和现价，算还要多少年回本 |
| 7 | 散户心态进化史 | 根据入市年份生成血泪进化阶段 |
| 8 | 每日股评嘴炮生成器 | 大师预测，次日自动打脸 |
| 9 | 持仓墓碑管理器 | 把亏钱的持仓当墓碑展示 |
| 10 | 市场情绪温度计 | 6维度综合计算市场情绪 |
